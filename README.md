## 🚀 Getting Started

Follow these steps to run the project locally:

**1. Clone the repository**
```bash
git clone https://github.com/Kaushik1014/postify.git
```

**2. Move into the project folder**
```bash
cd postify
```

**3. Install dependencies**
```bash
npm install
```

**4. Start the server**
```bash
nodemon index.js
```

**5. Open in your browser**
