## 🚀 Postify

Postify is a simple and interactive web application that allows users to create, view, edit, and delete posts.  
It demonstrates core **CRUD (Create, Read, Update, Delete)** functionality with a clean and beginner-friendly design.
This project is ideal for learning **Node.js**, **Express**, REST APIs, templating, and deployment.

---

## 🌐 Live Demo

👉 https://postify-9q78.onrender.com/posts

---

## ✨ Features

- 📄 View all posts
- ➕ Create a new post
- ✏️ Edit existing posts
- 🗑️ Delete posts
- ⚡ Clean and minimal UI
- 🌍 Deployed on Render

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, EJS
- **Backend:** Node.js, Express.js
- **Database:** In-memory data / JSON
- **Deployment:** Render
- **Version Control:** Git & GitHub

## 🚀 Getting Started (Run Locally)

1️⃣ Clone the repository
git clone https://github.com/Kaushik1014/postify.git

2️⃣ Move into the project folder
cd postify

3️⃣ Install dependencies
npm install

4️⃣ Start the server
nodemon index.js

5️⃣ Open in browser
http://localhost:3000/posts

## 🔗 API Endpoints

| Method | Endpoint   | Description       |
| ------ | ---------- | ----------------- |
| GET    | /posts     | Get all posts     |
| POST   | /posts     | Create a new post |
| GET    | /posts/:id | Get post by ID    |
| PUT    | /posts/:id | Update a post     |
| DELETE | /posts/:id | Delete a post     |

## 📌 Usage

1)Open the application

2)Click New Post to add content

3)Use Edit to update a post

4)Use Delete to remove a post
